Get started

Ensure you have Node installed in your system, then

```git clone https://github.com/jroge-uagrm/consultation-forum.git```

```cd consultation-forum```

```npm i```

```cp .env.example .env```

Develop start

```npm run-script dev-start```


Production start

```npm start```
